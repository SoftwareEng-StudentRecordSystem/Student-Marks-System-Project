There are two folders in this Repository, a Lab folder which has Laboratory exercises that are regularly updated based on the given exercise diliverables and the Project folder which is meant to have Front-end and Back-end development files as well as a project requirements documentation that will be continously updated throughout the project. 

To run the latex document you will need:
*The Latex doc file
*Latex Class file
*Latex Style file (KJN)
*Bibtex file
*BST file
*All used pictures
